# Apollo

Sistema de gerenciamento de chamados e suporte técnico desenvolvido para centralizar atendimento,
 organização de tickets e acompanhamento de solicitações.

## Funcionalidades
- Autenticação JWT
- Abertura de chamados
- Controle de status
- Prioridades
- Comentários
- Upload de anexos
- Dashboard administrativo
- Gestão de usuários

## Tecnologias

### Backend
- Python
- Django
- Django REST Framework
- PostgreSQL

### Frontend
- React
- Vite
- Tailwind CSS
